# Staff Report: HR Data Analysis Dashboard

## Project Overview
This project analyzes HR data to provide insights into employee demographics, salary distributions, and hiring trends using Excel and Power BI.

## Tools Used
* **Excel:** Data cleaning and preprocessing.
* **Power BI:** Data visualization and interactive dashboarding.
